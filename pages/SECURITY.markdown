## اقدامات امنیتی اعمال‌شده
برای اطمینان از امنیت سایت، اقدامات زیر پیاده‌سازی شده‌اند:
- **HTTPS**: فعال برای تمام صفحات از طریق GitHub Pages.
- **هدرهای امنیتی**: استفاده از متا تگ‌ها در کامپوننت `security` برای تنظیم:
  - `X-Frame-Options: SAMEORIGIN` (جلوگیری از کلیک‌جکینگ).
  - `X-XSS-Protection: 1; mode=block` (فیلتر XSS).
  - `X-Content-Type-Options: nosniff` (جلوگیری از حدس MIME).
  - `Content-Security-Policy` (محدود کردن منابع به دامنه‌های معتبر مثل `talakub.com` و CDNهای مجاز).
- **رمزگذاری ساده**: امکان فعال‌سازی رمز در `components/security/security.js` برای محدود کردن دسترسی در توسعه.
